Wodel-Test HTML Report

This page mirrors the four Wodel-Test Eclipse views: the global graphical summary, class results, mutator results, and test results. It supports one or many test suites, preserves the original filters, and uses the same green/red/grey semantics for detected, undetected, and error states.

Expected JSON shape

Top level: projectName, optional generatedAt, and testSuites[]. Each suite may contain global, classResults[], mutatorResults[], and testResults[].

The report data is preloaded automatically. You can still paste or open another JSON file if you want to inspect a different run.
